    "# GroupBy\n",
    "\n",
    "\"Group by\" operations refers to an implementation of the \"split-apply-combine\" approach known from [pandas](https://pandas.pydata.org/pandas-docs/stable/user_guide/groupby.html) and [xarray](http://xarray.pydata.org/en/stable/groupby.html).\n",
    "We currently support only a limited number of operations that can be applied.\n",
    "\n",
    "## Grouping with bins\n",
    "\n",
    "Note that this notebooks requires [Mantid](https://www.mantidproject.org/Main_Page) and data files.\n",
    "A [binder](https://mybinder.org/v2/gh/scipp/scipp-neutron-jupyter-demo/main) is available that can run this notebook.\n",
    "\n",
    "[PG3_4844_event.nxs](http://198.74.56.37/ftp/external-data/MD5/d5ae38871d0a09a28ae01f85d969de1e) is required for this notebook.\n",
    "\n",
    "Rename the files upon download."
   ]

   "source": [
    "### Example 1 (dense data): split-sum-combine\n",
    "\n",
    "We histogram the event data:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "bins = sc.Variable(['tof'], values=np.arange(0.0, 17000.0, 50.0), unit=sc.units.us)\n",
    "pos_hist = sc.histogram(events, bins)"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "A plot shows the shortcoming of the data representation.\n",
    "There is no physical meaning attached to the \"spectrum\" dimension and the plot is hard to interpret:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "sc.plot(pos_hist)"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "To improve the plot, we first store the scattering angle as labels in the data array.\n",
    "Then we create a variable containing the desired target binning:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "pos_hist.coords['two_theta'] = scn.two_theta(pos_hist)\n",
    "two_theta = sc.Variable(['two_theta'],\n",
    "                        unit=sc.units.rad,\n",
    "                        values=np.linspace(0.0, np.pi, num=500))"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "We use `scipp.groupby` with the desired bins and apply a `sum` over dimension `spectrum`:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "theta_hist = sc.groupby(pos_hist, 'two_theta', bins=two_theta).sum('spectrum')"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [

   ]

  {
   "cell_type": "code",
   "execution_count": null,

   "outputs": [],

   ]
  },
